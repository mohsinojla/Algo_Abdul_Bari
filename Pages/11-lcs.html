Longest Common Subsequence (LCS)

Find the longest subsequence present in both strings (not necessarily contiguous).

▶ Watch Abdul Bari on YouTube
📖 Concept Overview

A subsequence is a sequence derived from a string by deleting some characters without changing order. E.g., "ACE" is a subsequence of "ABCDE".

LCS Problem: Given strings X and Y, find the length of their longest common subsequence.

if X[i] == Y[j]: dp[i][j] = dp[i-1][j-1] + 1
else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
Base: dp[0][j] = dp[i][0] = 0

After filling the DP table, trace back from dp[m][n] to find the actual subsequence:

  • If X[i]==Y[j]: include this character, go diagonal ↖
  • Else: go in direction of larger value (up ↑ or left ←)
🔑 Key Points
1. LCS is NOT necessarily contiguous (unlike Longest Common Substring).
2. DP table size: (m+1) × (n+1) where m, n are string lengths.
3. First row and column are all zeros (base cases).
4. Time: O(m×n), Space: O(m×n).
5. Traceback gives the actual LCS string (read diagonal moves).
6. There can be multiple LCS of the same length.
7. LCS length = dp[m][n] (bottom-right cell).

TableX="AGGTAB"(6), Y="GXTXAYB"(7). dp is 7×8.
Fill row by row:
Row A(1): matches Y[5]=A → dp[1][5]=1, rest propagate
Row G(2): matches Y[1]=G → dp[2][1]=1, continues...
Row G(3): matches Y[1]=G → dp[3][1]=1
Row T(4): matches Y[3]=T,Y[5]=X(no),Y[3]=T yes...
Row A(5): matches Y[5]=A
Row B(6): matches Y[7]=B
Resultdp[6][7] = 4
Traceback: B←A←T←G → LCS = "GTAB"
✅ LCS = "GTAB", length = 4

Q4 What is the difference between LCS (Longest Common Subsequence) and LCS (Longest Common Substring)? Give an example. ▼
SubsequenceSubsequence: Characters don't need to be contiguous but must maintain relative order.
X="ABCDE", Y="ACE" → LCS = "ACE" (length 3)
Recurrence: if match → dp[i-1][j-1]+1, else max(dp[i-1][j], dp[i][j-1])
SubstringSubstring: Characters must be contiguous in BOTH strings.
X="ABCDE", Y="BCDE" → Longest Common Substring = "BCDE" (length 4)
Recurrence: if match → dp[i-1][j-1]+1, else 0
ExampleX="ABAB", Y="BABA"
LCS (subsequence) = "ABA" or "BAB" → length 3
LCS (substring) = "ABA" or "BAB" → length 3 (coincidence here)
X="ABCDEF", Y="FBDAMCE": subseq LCS=3, substring LCS=1
✅ Subsequence: non-contiguous, order preserved. Substring: contiguous block. Subsequence ≥ Substring in length.
